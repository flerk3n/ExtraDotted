How it works

Pixel → circle, not image → tag

Reads brightness from each pixel of your source and renders it as a sized circle on canvas. Works with photos, logos, text screenshots any visual content.

Accessibility support

Braille & tactile graphics

Export dot patterns as ASCII/Unicode for braille conversion or high-resolution PNG for tactile printing. Adjustable dot spacing matches embosser specifications.

Why it stays responsive

Redraws on every resize

On viewport change the canvas recalculates dimensions and dot density, so the portrait always fills the stage at the right resolution no blurring, no cropping.

Deployment path

Export SVG, drop it in

The SVG export produces a self-contained vector file of circles you can inline directly into any hero section. No canvas dependency, no JavaScript required after export.

Universal input

Any image, any text

Upload photographs, illustrations, logos, or screenshots of text. The dot renderer adapts to any visual content, making it accessible and artistic.

Screen reader friendly

Assistive tech compatible

ASCII output works seamlessly with screen readers and assistive technologies. High-contrast modes ensure visibility for low-vision users.
